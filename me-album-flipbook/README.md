# Album Flipbook v1.3

## Changes
- The `docker-compose.yml` is hardcoded with `me-album-flipbook_server_1` proxy host.
- The `image` is hardcoded to `ghcr.io/your-github-user/album-flipbook:1.3.0` (Change this!).
- Admins now access settings via `http://umbrel.local:3017/#admin` or clicking "Admin login" in the footer.
- Clients do not see any admin UI or buttons.
- Admins can configure Page Width and Page Height (A4 Portrait, A4 Landscape, Square, A5).
